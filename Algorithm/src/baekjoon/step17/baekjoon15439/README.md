#조합론 베라의 패션
<p>
조합론 15439 문제 
</p>
### 요구사항

베라는 상의 N벌과 하의 n벌이 있다 i번째 상의와 i번째 하의는 모두 색상 i를 가진다 n개의 색상은 모두 서로 다르다 </br>
상의와 하의가 서로 다른 색상인 조합은 총 몇개가 일까

### 입력
N </br>
5

### 출력
20

### 제한
N은 정수이다
1<=N<=2017

### 분석
상의와 하의 조합을 구하는 문제
n*n 겹치는 조합을 제외해야 하기 때문에
n*n-n 혹은 n*(n-1) 
5을 입력했을때 5*(5-1) = 20
25-5= 20
